# Python学生宿舍管理系统 python183

## 项目概述

本项目是基于Python Flask框架开发的学生宿舍管理系统，集成了宿舍分配、保修、查询以及学生和宿舍信息管理等功能，旨在提高宿舍管理的效率和准确性。

## 技术栈

- **后端语言与框架**：Python 3.0以上，Flask
- **前端技术**：HTML、CSS、JavaScript、jQuery
- **数据库**：MySQL 8.0
- **开发工具**：PyCharm、Navicat

## 功能模块

- **宿舍分配**：实现宿舍的分配与管理功能
- **宿舍保修**：提供在线报修功能
- **宿舍查询**：方便用户查询宿舍信息
- **学生信息管理**：管理学生基本信息
- **宿舍信息管理**：维护宿舍基本信息
- **通知管理**：发布和管理通知公告

## 系统角色

- 管理员：负责系统维护、信息管理、通知发布等
- 学生：使用系统进行宿舍查询、保修等

## 运行环境

- Python 3.0以上
- MySQL 8.0
- 需要安装相关Python依赖包（见项目文档）

## 部署步骤

1. 安装Python环境和MySQL数据库
2. 克隆或下载项目源代码
3. 安装项目所需的Python依赖包
4. 配置数据库连接信息
5. 运行项目

详细步骤见项目文档。
